# assets

Сюда кладётся иконка приложения: **`app.ico`**.

- Формат: `.ico` (Windows icon).
- Желательно несколько размеров в одном файле: 16×16, 32×32, 48×48, 256×256.
- Как только файл `assets/app.ico` появится, сборка (`build.ps1` / `make`)
  автоматически вкомпилирует его в `cert-migrator.exe` — Explorer и панель
  задач покажут иконку. Пока файла нет, exe собирается без иконки.
